Steps and Challenges involved in making this project

  • Planning

    - I chose EJS for frontend because I was unaware of react at that time
    - For the backend, I decided to go with Node and try postgresql

  • Setting up the project

    - Downloading the required google fonts and images
    - Organizing frontend and backend files
    - Initializing npm on backend

  • Code Execution

    - Implemented html,css and js for overall page functionality

    - Added EJS in html as to make the page dynamic
    - Created an express server
    - Added get and post routes to receive and send data
    - Build a database in postgresql
    - Created a table for storing the bookmark ebooks
    - Implemented routes on displaying the eBooks on page load
    - Fixed bugs
    - Testing code and optimizing

  • Challenges Faced

    - Responsive design
