Steps and Challenges involved in making this project

  • Planning

    - I planned to make this project with only React without need of backend as the user is just going to look into news

  • Setting up the project

    - Downloaded the required google fonts with different weights
    - Organized the folder for different components and pages for frontend and backend
    - Created React app using npx
    -Testing the api in postman

  • Code Execution

    - Created multiple components for each section
    - Exported and imported the components to main file
    - Designed the page elements using vanilla css and used material-ui for the icons
    - Made API calls using fetch api as to get details regarding each news
    - Used React router dom for routing between components
    - Fixed bugs
    - Testing code and optimizing

  • Challenges Faced

    - Implementing the "see more" functionality
