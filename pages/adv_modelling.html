Organic and inorganic sculpts

Project Demo

Steps and Challenges involved in making this project

  • Planning

    - Creating rough sketches of the 3D models for visualization
    - Collecting reference images for 3D models

  • Setting up the project

    - Creating Zbrush files for organic models and Maya files for inorganic models

  • Execution

    - Studying each model in order to make it in an efficient way
    - Sculpting organic models using Zbrush brushes
    - Using Maya for modelling inorganic models
    - Substance painter for painting and adding texture to the model
    - Arranging the models in an environment
    - Installing lights in the environment
    - Rendering the environment in multiple angles

  • Challenges Faced

    - Having a proper mesh topology
    - Using minimum poly count to make it lightweight
    - Choosing the right brush for the model
    - Matching the 3D model to the real life object
    - Painting the model to make it look realistic and proper
