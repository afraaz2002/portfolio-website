Steps and Challenges involved in making this project

  • Planning

    - I got the idea of creating a recipe app from developedByEd
    - I planned of having seperate sliders for different sections
    - I really wanted to try React so this was my first project with react and decided to use mongoDB and node for the whole app

  • Setting up the project

    - Downloaded the required google fonts with different weights
    - Chose a matching background and color combination for the app
    - Organized the folder for frontend and backend
    - Created React app using npx
    - Initialized npm for the backend
    -Testing the api in postman

  • Code Execution

    - Created multiple components for each section
    - Exported and imported the components to main file
    - Designed the page elements using styled component and other react packages
    - Made API calls using fetch api as to get details regarding each recipes
    - Installed packages such as express,mongodb,mongoose,body-parser,etc
    - Created an express server
    - Initialized mongoose
    -Created database and schema
    - Added get and post routes to receive and send data
    - Fixed bugs
    - Testing code and optimizing

  • Challenges Faced

    - Modern Search bar design
    - Handling data stored in database
    - Code Optimization
