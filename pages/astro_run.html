Space theme action filled mobile game

Beware of toxic gas which exhaust your fuel

Project Demo

Astro Run is a 2D side-scrolling action game where players take on the role of an astronaut running through space. The game features various obstacles and enemies that the player must avoid while collecting coins and power-ups to increase their score and improve their abilities. With simple controls and addictive gameplay, Astro Run offers a fun and challenging experience for gamers of all ages.

Steps and Challenges involved in making this project

  • Planning

    - Made a game design document to plan out every aspects of the game
    - The game design document contains details such as level design, characters and backgrounds, etc

  • Setting up the project

    - Initializing the Unity project
    - Wireframing the GUI, levels and characters
    - Organizing the folders for different assets such as the sprites, C# scripts, materials, etc

  • Execution

    - Designing sprites in adobe photoshop
    - Importing the assets in Unity
    - Rendering the assets to the scene
    - Creating mulitple scenes for multiple levels
    - Creating game manager script which controls the entire game
    - Building the player controller
    - Adding physics for player and game elements
    - Creating tons of other scripts for other functionalites which includes level progression, shooting, black hole pull effect, etc
    - Implementing GUI which is responsive to all devices
    - Fixing bugs and improving the code
    - Enhancing game performance
    - Continous testing and debugging
    - Exporting the game as an apk file
    - Launching the game on playstore

  • Challenges Faced

    - Player controls
    - Optimizing/Refactoring the code
    - Maintaining the game
    - Fixing bugs and glitches
